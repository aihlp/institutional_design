# From Rules to Behavioral Clusters

## Summary

The core proposal: **Institutions should be defined as empirically distinguishable, statistically stable clusters of behavioral trajectories that arise from a spectrum of possible reactions to an identifiable informational signal.**

This is not merely replacing the word "rule" with "cluster"—it changes the **epistemological status** of the analytical object.

## The Three Primitives

### 1. Behavioral Trajectory

**Definition**: The time-ordered sequence of reactions produced by a single agent in response to a stream of identifiable informational signals.

| Classical Concept | New Concept |
|-------------------|-------------|
| "Behavior" (vague, inferred) | Behavioral trajectory (fully specified) |
| Unobservable intentions | Timestamped, observable actions/inactions |
| Mental states | Digital trace data |

**Key feature**: Individual-level analogue of behavior, fully specified by logged actions.

### 2. Behavioral Cluster

**Definition**: A grouping of similar behavioral trajectories identified through unsupervised learning techniques.

**Critical points**:
- Clusters are **not assumed**; they are **constructed** by the analyst
- Uses transparent, reproducible criteria
- Example criterion: minimum divergence threshold between behavioral distributions in adjacent time windows
- A cluster is what the analyst identifies as sufficiently stable and distinct to warrant the label "institution"

### 3. Behavioral Spectrum

**Definition**: The full distribution of possible reactions to a given informational signal within a defined population at a given time.

**Function**: 
- Captures range of behavioral variation **before** clustering
- Serves as baseline against which stability and distinctiveness of clusters can be assessed

## Epistemological Shift

| Classical Framework | Proposed Framework |
|---------------------|-------------------|
| Institution = **inferred from** behavior | Institution = **identified in** behavior |
| Latent cause behind regularity | Pattern clustered, bounded, tracked directly |
| Requires unobservable mediating constructs | Requires only observable data |
| Ontological discovery | Methodological construction |

## Concrete Example: Tax Compliance

### Classical View
"Tax compliance" is an institution because:
- A rule exists (the tax code)
- The rule constrains behavior
- Institution = the rule

### New Framework View
"Tax compliance" is a stable behavioral cluster:
- Set of individual trajectories converging on same observable pattern
- Pattern: filing returns, reporting income, making payments
- Response to identifiable signals: tax code, enforcement communications, social expectations
- **Identified by statistical properties**, not by supposed causal rule

**Critical distinction**: Whether a rule exists, and whether anyone believes in it, are **separate empirical questions** that may or may not be relevant to the cluster's persistence.

## Why This Dissolves the Circularity

Classical problem:
```
Institutions explain behavior → But institutions inferred from behavior → Tautology
```

New framework:
```
Institutions ARE behavioral patterns → Measured like any statistical regularity → No circularity
```

### Measurement Requirements

| What Classical Theory Needs | What New Framework Needs |
|----------------------------|-------------------------|
| Access to beliefs/intentions | Access to behavioral logs |
| Interpretation of meanings | Clustering algorithms |
| Inference of latent constructs | Stationarity tests |
| Normative analysis | Transition matrices |

**Key insight**: Measurement does not depend on anyone's beliefs, intentions, or interpretations. It depends only on the data.
