# Tier 1: Base Definitions

## Overview

This tier establishes the foundational definitions and primitives that form the basis of the behavioral cluster framework for institutional analysis.

## Core Components

### The Five Primitives

Tier 1 defines five fundamental concepts that serve as the building blocks for all higher-level analysis:

1. **Behavioral Cluster** - Empirically identified groupings of behaviorally similar subjects
2. **Informational Signal** - Platform-mediated exposure to content that influences user behavior
3. **Reaction** - Observable user response to informational signals
4. **Behavioral Trajectory** - Temporal sequence of reactions forming a pattern
5. **Behavioral Spectrum** - Range of possible behavioral trajectories within a population

## Purpose

These base definitions provide:
- Clear, operationalizable concepts for empirical research
- A common vocabulary for cross-disciplinary collaboration
- Foundation for mathematical formalization in Tier 3
